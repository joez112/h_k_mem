# h_k_mem - 分层记忆宫殿架构

## 🎯 概述

**h_k_mem** 是一个突破Hermes Agent 2,200字符记忆限制的创新架构。通过分层记忆宫殿设计，将详细数据存储在外部文件中，记忆只保留索引，实现35倍以上的存储扩展。

## 📊 核心优势

- **突破限制**: 从2,200字符扩展到12,839+字符（35倍扩展）
- **结构化组织**: 五层建筑+地下室分层架构
- **快速访问**: 索引系统提供快速导航
- **易于维护**: 各层独立，互不干扰
- **可扩展性**: 随时添加新的记忆文件

## 🏛️ 架构设计

### 记忆宫殿索引系统
```
记忆宫殿索引系统
├── 1F - 项目任务索引 → /记忆宫殿/1F_项目任务区.md
├── 2F - 系统架构索引 → /记忆宫殿/2F_系统架构区.md
├── 3F - 数据分析索引 → /记忆宫殿/3F_数据分析区.md
├── 4F - 用户要求索引 → /记忆宫殿/4F_用户要求区.md
├── 5F - 文件管理索引 → /记忆宫殿/5F_文件管理区.md
└── B1 - 记忆管理索引 → /记忆宫殿/B1_记忆管理区.md
```

### 访问流程
1. **查看索引** → 读取记忆宫殿索引
2. **访问详细** → 根据索引读取对应记忆文件
3. **更新内容** → 修改记忆文件
4. **同步索引** → 必要时更新索引引用

### 1. 初始化记忆宫殿

from hermes_tools import write_file, memory

memory_index = """🏛️ 记忆宫殿索引系统
===================

🔑 访问：索引→文件→详细内容
📊 原则：索引化、分层化、文件化

📁 索引结构：

2F - 系统架构索引 → /记忆宫殿/2F_系统架构区.md  
3F - 数据分析索引 → /记忆宫殿/3F_数据分析区.md
4F - 用户要求索引 → /记忆宫殿/4F_用户要求区.md
5F - 文件管理索引 → /记忆宫殿/5F_文件管理区.md
B1 - 记忆管理索引 → /记忆宫殿/B1_记忆管理区.md

🔗 访问方式：
1. 查看索引 → 记忆宫殿索引
2. 查看详细 → 读取对应记忆文件
3. 更新内容 → 修改记忆文件

## 🔧 高级功能

### 自动初始化脚本
```python
def init_memory_palace():
    """初始化完整的记忆宫殿"""
    import os
    from hermes_tools import write_file, memory
    
    # 创建目录
    os.makedirs("记忆宫殿", exist_ok=True)
    
    # 设置索引
    index_content = """🏛️ 记忆宫殿索引系统
===================
📍 架构：五层建筑+地下室（索引层）
🔑 访问：索引→文件→详细内容
📊 原则：索引化、分层化、文件化

📁 索引结构：
1F - 项目任务索引 → /记忆宫殿/1F_项目任务区.md
2F - 系统架构索引 → /记忆宫殿/2F_系统架构区.md  
3F - 数据分析索引 → /记忆宫殿/3F_数据分析区.md
4F - 用户要求索引 → /记忆宫殿/4F_用户要求区.md
5F - 文件管理索引 → /记忆宫殿/5F_文件管理区.md
B1 - 记忆管理索引 → /记忆宫殿/B1_记忆管理区.md

🔗 访问方式：
1. 查看索引 → 记忆宫殿索引
2. 查看详细 → 读取对应记忆文件
3. 更新内容 → 修改记忆文件
4. 同步索引 → 更新索引引用"""
    
    memory(action='add', target='memory', content=index_content)
    
    # 创建各层文件
    layers = {
        "1F_项目任务区.md": "# 🏢 1F - 项目任务区 (入口大厅)\n...",
        "2F_系统架构区.md": "# 🏢 2F - 系统架构区 (左翼)\n...",
        "3F_数据分析区.md": "# 🏢 3F - 数据分析区 (右翼)\n...",
        "4F_用户要求区.md": "# 🏢 4F - 用户要求区 (二楼)\n...",
        "5F_文件管理区.md": "# 🏢 5F - 文件管理区 (三楼)\n...",
        "B1_记忆管理区.md": "# 🏢 B1 - 记忆管理区 (地下室)\n..."
    }
    
    for filename, content in layers.items():
        write_file(f"记忆宫殿/{filename}", content)
    
    return "记忆宫殿初始化完成"
```

## 🔍 故障排除

### 常见问题
1. **文件不存在错误**
   - 检查记忆宫殿目录是否存在
   - 运行初始化脚本

2. **索引不同步**
   - 运行sync_memory_index()函数
   - 手动更新记忆索引

3. **容量仍然不足**
   - 创建新的记忆层（如6F、7F）
   - 进一步拆分详细数据
